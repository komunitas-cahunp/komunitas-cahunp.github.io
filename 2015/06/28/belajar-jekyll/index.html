Belajar Jekyll

Ikut berkontribusi menulis blog di KCU

Overview

Tepat di waktu-waktu ngabuburit, anak-anak komunitas lagi ngumpul untuk belajar jekyll, apa itu jekyll? sudah dibahas di post Ngopinya Komunitas. Nah, karena blog komunitas dibuat menggunakan jekyll, maka sebagai asumsi untuk belajar jekyll, dengan cara ikut menulis blog di Komunitas CAH UNP. Yang perlu disiapkan adalah:

  • Daftar akun Github
  • Buka Repository, selanjutnya Forking ke akun pribadi anda.
  • Generating SSH-Keygen dan ikuti panduan hingga selesai.
  • Buka Git Bash:
  $ git clone git@github.com:<username>/komunitas-cahunp.github.io.git
  $ cd komunitas-cahunp.github.io/
  $ touch _post/tahun-bulan-tanggal-judul-artikel.md
  $ sublime _post/tahun-bulan-tanggal-judul-artikel.md
